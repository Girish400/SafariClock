An ultra-professional Flip Clock PWA optimized for iPhone StandBy mode.

### Features
- OLED Pure Black / Burn-in protection.
- Always-on display via Wake Lock.
- Automatic Night Mode (Red tint).
- Fullscreen PWA support.

### Setup Instructions
1. Upload all files to a repo named `SafariClock`.
2. Go to **Settings > Pages**.
3. Set the source to the `main` branch and `/root`.
4. Open the link in **Safari** on iPhone.
5. Tap the **Share** button > **Add to Home Screen**.
